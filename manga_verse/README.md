# MangaVerse — Flutter Android App

A production-ready manga/webtoon reader with 12 original stories, dark anime UI, AdMob integration, and offline support.

---

## Project Structure

```
manga_verse/
├── assets/data/          # stories.json + chapters.json (12 stories, 60 chapters)
├── lib/
│   ├── core/             # theme, router, constants, utils
│   ├── data/             # models, datasources, repository impls
│   ├── domain/           # entities, repository interfaces
│   └── presentation/     # providers (Riverpod), screens, widgets
├── android/              # Android config with AdMob
└── pubspec.yaml
```

---

## How to Run

### Prerequisites
- Flutter SDK >= 3.1.0
- Android Studio / VS Code
- Android device or emulator (API 21+)

### Steps

```bash
# 1. Navigate into project
cd manga_verse

# 2. Install dependencies
flutter pub get

# 3. Run on connected device / emulator
flutter run
```

> **First-time setup:** The android/ folder here replaces the one generated by `flutter create`.
> If you used `flutter create manga_verse` separately, copy only the `lib/` and `assets/` folders
> plus `pubspec.yaml`, then follow the Android setup below.

## Build Release AAB (for Play Store)

```bash
# 1. Create a keystore (one-time)
keytool -genkey -v -keystore ~/upload-keystore.jks \
  -keyalg RSA -keysize 2048 -validity 10000 \
  -alias upload

# 2. Create android/key.properties
storePassword=<your-store-password>
keyPassword=<your-key-password>
keyAlias=upload
storeFile=<absolute-path-to>/upload-keystore.jks

# 3. Update android/app/build.gradle to reference key.properties (see Flutter docs)

# 4. Build AAB
flutter build appbundle --release
# Output: build/app/outputs/bundle/release/app-release.aab
```

## AdMob Setup

The app currently uses **Google's official test Ad Unit IDs** — safe for development, but earns no real revenue.

### Replace with Real IDs Before Publishing

1. Create an AdMob account at https://admob.google.com
2. Create an Android app in AdMob
3. Create three ad units: Banner, Interstitial, Rewarded

4. Open `lib/core/constants/ad_constants.dart`:

```dart
static const String appId = 'ca-app-pub-XXXXXXXXXXXXXXXX~XXXXXXXXXX';
static const String bannerAdUnitId = 'ca-app-pub-XXXXXXXXXXXXXXXX/XXXXXXXXXX';
static const String interstitialAdUnitId = 'ca-app-pub-XXXXXXXXXXXXXXXX/XXXXXXXXXX';
static const String rewardedAdUnitId = 'ca-app-pub-XXXXXXXXXXXXXXXX/XXXXXXXXXX';
```

5. Open `android/app/src/main/AndroidManifest.xml` and replace the `APPLICATION_ID` meta-data value with your real App ID.

### Ad Placements
| Ad Type | Placement |
|---|---|
| Banner | Bottom of Home screen |
| Interstitial | After reaching end of a chapter |
| Rewarded | Unlock bonus (locked) chapters |

## Features

| Feature | Details |
|---|---|
| **Stories** | 12 original manga stories, 5 chapters each |
| **Reader** | Vertical scroll, progress saving, chapter nav |
| **Offline** | Images cached via `cached_network_image` after first load |
| **Favorites** | Persistent across sessions (Hive) |
| **History** | Last-read chapter + progress bar + timestamps |
| **Search** | Debounced, searches title/tags/genre |
| **Categories** | Filter by 10 genres |
| **Onboarding** | 3-page intro shown once |
| **Ads** | Banner + Interstitial + Rewarded (AdMob) |
| **Theme** | Dark anime style, glassmorphism, Hero animations |

## Architecture

- **Clean Architecture** — domain / data / presentation layers
- **State management** — Riverpod 2.x (`AsyncNotifierProvider`, `NotifierProvider`)
- **Navigation** — go_router 13.x with `StatefulShellRoute` for bottom nav
- **Storage** — Hive (favorites, history, progress) + SharedPreferences-compatible
- **Images** — `cached_network_image` (auto-cached for offline)
- **Animations** — `flutter_animate` on cards, screens, onboarding

## App Icon

To set a custom app icon, place your icon image in `assets/images/app_icon.png`
(1024×1024 recommended) and run:

```bash
flutter pub add flutter_launcher_icons --dev
```

Add to `pubspec.yaml`:
```yaml
flutter_launcher_icons:
  android: true
  image_path: "assets/images/app_icon.png"
```

Then:
```bash
flutter pub run flutter_launcher_icons
```
